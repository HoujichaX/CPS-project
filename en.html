Basics

Pack it, open it

Two moves cover most days. They feel the same in every edition.

☕ Pack (compress)

  1. On the Compress tab, pick a file (drag & drop works)
  2. Choose a drink from the menu (Water / Ice Cocoa / …)
  3. Press Pack
  4. Choose where to save — done (.cps)

🫗 Open (unpack)

  1. On the Unpack tab, pick a .cps / .cpsv
  2. Press Unpack
  3. The method is auto-detected — your original file comes back
  4. Damaged parts are repaired with RS error correction when possible
Before using Hot Cocoa / Coffee: install a .cpsdict via Recipe Shelf → Add recipe. Layers are stored under %APPDATA%\cps\layers.

How-To

Step by step, tab by tab

Following the sidebar. Preview has Compress / Unpack / Pairs; Standard adds Recipe Shelf and Receive; Pro adds Workshop, Friend Key and Recipe Pack.

Compress all editions

  1. Pick a file — Browse, or drag & drop onto the window (folders too).
  2. Pick a drink — from the tabs at the top. Ice Cocoa picks a built-in dictionary; Hot Cocoa / Coffee / Shake pick an installed recipe.
  3. For precious data, tick Rugged pack (CPSV) — a frame container protected by RS.
  4. Press Pack, choose a destination, and you get a .cps (or .cpsv).
📷Screenshot: Compress tab with method tabs and a file selected

Unpack all editions

  1. Pick a .cps / .cpsv (drop works).
  2. Press Unpack — the method is auto-detected, even if the file was renamed.
  3. Rugged packs attempt RS repair when parts are missing or corrupted.
📷Screenshot: Unpack tab with a completed log

Recipe Shelf Standard / Pro

  1. Add recipe and choose a .cpsdict to install (stored in %APPDATA%\cps\layers).
  2. The list shows what's installed — it feeds the sub-selection for Hot Cocoa / Coffee / Shake.
  3. Standard ships with sample_layers\ (four Japanese part-of-speech layers) to try right away.
📷Screenshot: Recipe Shelf with a few entries

Receive Standard / Pro

  1. Self-contained pack (.cprp) — just open it. The recipe travels inside; nothing to install.
  2. Friend Key — point at the menu file and the .cpmn, then unpack.
📷Screenshot: Receive tab

Pairs all editions (tiered)

  1. Pro: register a new pair → brew a dictionary from samples to grow a new generation → export a generation pack (.cpgen) for your partner.
  2. Standard: import the .cpgen, then pack with the pair dictionary — Efficient .cpp2 / Secret .cpp3 / Keyed .cpp3l.
  3. Preview: import generations and open pair packs.
  4. Opening looks up the right dictionary by lineage ID and generation — old packs still open with old generations.
📷Screenshot: Pairs tab with one entry in the ledger

Workshop · Friend Key · Recipe Pack Pro

  1. Workshop: brew Hot Cocoa / Coffee / Shake recipes from sample files into .cpsdict. Optional LLM assistance (bring your own key).
  2. Friend Key: create a limited-release pack (.cpmn + menu).
  3. Recipe Pack: create a self-contained .cprp — receivers open it with nothing installed.
📷Screenshot: Workshop with a finished brew log

Three ways to send

Efficient .cpp2

The everyday cup — smallest size with your pair dictionary.

Secret .cpp3

Substitution + 4D shuffle hides content and patterns. No key exchange needed — the pair dictionary is the first key.

Keyed .cpp3l

Secret plus a per-message key on top. Only someone with both the dictionary and the key can open it. Hand the key over separately.

Secret is not encryption. It is limited disclosure — hiding plaintext and patterns. For a message you truly want locked down, use Keyed (.cpp3l) and pass the key another way. Creating pairs and growing dictionaries is Pro; sending with a received dictionary is Standard; importing and opening works even in the free Preview.

File types

Extension cheat sheet

  • .cpsStandard pack — one of the five methods, auto-detected on open.
  • .cpsvRugged pack. A frame + RS container built to survive damage.
  • .cpsdictLayer / dictionary. Install from the Recipe Shelf.
  • .cpgenPair generation pack — hands a grown dictionary to your partner (lineage + generation inside).
  • .cpp2Pair pack (Efficient). Smallest everyday pack with the pair dictionary.
  • .cpp3Pair pack (Secret). Substitution + 4D shuffle, limited disclosure.
  • .cpp3lPair pack (Keyed). Secret plus a per-message key — two locks.
  • .cprpSelf-contained pack with the recipe inside. Opens with nothing installed.
  • .zdictBuilt-in Ice Cocoa dictionaries (six, bundled).
