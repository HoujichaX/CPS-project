データを、一杯に
詰めるカフェ。

CPS はファイルを「飲みもの」に見立てて圧縮する Windows アプリです。 辞書のいらない一杯から、こだわりの一杯まで。壊れにくい瓶づめや、フレンドとの受け渡しもできます。

Preview を入手 メニューを見る Preview は無料・Microsoft Store 公開中

メニュー

5つの詰め方から選ぶ

どれも「ファイルを選ぶ → 一杯を選ぶ → 詰める」だけ。辞書やレシピが要る一杯は、 先に「レイヤー管理」でインストールしておきます。「こく」は圧縮の効き目のイメージで、実際はデータによって変わります。

使い方

詰める・開ける、基本の2つ

まずはこれだけ。どのグレードでも同じ手触りで動きます。

☕ 詰める(梱包)

  1. 「梱包する」タブでファイルを選ぶ(ドロップでも可)
  2. メニューから一杯を選ぶ(水 / アイスココア / …)
  3. 「梱包する」を押す
  4. 保存先を決めて完成(.cps)

🫗 開ける(開封)

  1. 「開封する」タブで .cps / .cpsv を選ぶ
  2. 「開封する」を押す
  3. 方式は自動判定 — 元のファイルに戻る
  4. 途中が欠けていてもRS 誤り訂正で復元を試みます
ココア / コーヒーを使う前に: 「レシピ棚 → レシピを追加」で .cpsdict をインストールしてください。 Standard には同梱の sample_layers\ (日本語の品詞別4種)が入っています。インストール先は %APPDATA%\cps\layers です。

瓶づめ

壊れにくい、頑丈な一本

大事なデータは「瓶づめ(CPSV)」に。フレーム構造と Reed–Solomon で守るので、一部が欠けても・化けても復元を試みます。

瓶づめは梱包タブで「瓶づめ」を選ぶだけ。外殻はフレーム単位で少しずつ書き込むので、巨大なファイルでも安定します。

Preview 版では内部の CPS を含めて 1.5 GiB までが対象です。開封は中身を見て自動判定するため、拡張子を変えても開けます。

ペアリング

二人で、育てていく辞書

特定の相手と「ペア」を組むと、やり取りするほど二人だけの辞書が育ちます。世代を重ねるごとに賢くなり、 同じ相手へのパックはどんどん小さくなっていく——それがペアリングです。

送り方は3つ

効率 .cpp2

ペア辞書でいちばん小さく詰める、日常づかいの一杯。

ひみつ .cpp3

置換＋4Dシャッフルで中身とパターンを伏せる。鍵の受け渡しは不要(ペア辞書が第一の鍵)。

かぎつき .cpp3l

ひみつの上に「1通ごとの鍵(フレンドkey)」を重ねた二層。辞書と鍵の両方を持つ人だけが開けます。鍵は別便で手渡し。

「ひみつ」は暗号ではありません。 平文とパターンを伏せる限定開示です。強く限定したい1通は「かぎつき(.cpp3l)」にして、鍵を別に渡してください。 ペアを作る・辞書を育てるのは Pro、受け取った辞書で送るのは Standard、受け取って開くだけなら無料 Preview でできます。

操作ガイド

タブごとの操作手順

サイドバーのタブに沿って。Preview は 圧縮/開封/ペア の3つ、Standard は＋レシピ棚/受け取り、Pro はさらにレシピ工房・鍵かけ・レシピパックが加わります。

圧縮 全グレード

  1. ファイルを選ぶ — 「参照」またはウィンドウへドラッグ&ドロップ(フォルダも可)。
  2. 方式を選ぶ — 上部のタブから一杯を選択。アイスココアは内蔵辞書、ココア/コーヒー/シェイクはインストール済みレシピをサブ選択。
  3. 大事なデータなら「瓶づめ(CPSV)」にチェック — RSで守る頑丈梱包に。
  4. 「梱包する」を押して保存先を決めると .cps(瓶づめなら .cpsv)が完成。
📷スクリーンショット: 圧縮タブ(方式タブとファイル選択が見える状態)

開封 全グレード

  1. .cps / .cpsv を選ぶ(ドロップ可)。
  2. 「開封する」を押す — 方式は中身から自動判定。拡張子が変わっていても開けます。
  3. 瓶づめは一部が欠けていても RS 誤り訂正で復元を試みます。
📷スクリーンショット: 開封タブ(完了ログが出た状態)

レシピ棚 Standard / Pro

  1. 「レシピを追加」で .cpsdict を選ぶとインストール(保存先: %APPDATA%\cps\layers)。
  2. 一覧で入っているレシピを確認。ココア/コーヒー/シェイクのサブ選択に反映されます。
  3. Standard には同梱の sample_layers\(日本語の品詞別4種)があります。まずこれを入れると試せます。
📷スクリーンショット: レシピ棚(一覧に数件入った状態)

受け取り Standard / Pro

  1. 自己完結パック(.cprp) — 選んで開くだけ。レシピ同梱なので何も入れずに開けます。
  2. フレンドkey — menu と .cpmn の2ファイルを指定して開封。
📷スクリーンショット: 受け取りタブ

ペア 全グレード(できることは段階制)

  1. Pro: 「新規ペア」で相手を登録 → 素材から辞書を焼いて世代を育てる → 世代パック(.cpgen)を書き出して相手へ。
  2. Standard: 受け取った .cpgen を取り込み、そのペア辞書で梱包(効率 .cpp2 / ひみつ .cpp3 / かぎつき .cpp3l)。
  3. Preview: 世代辞書を取り込んでペアパックを開くところまで。
  4. 開封は系譜ID/世代から保管庫の辞書を自動で引きます。古いパックも旧世代で開けます。
📷スクリーンショット: ペアタブ(台帳に1件ある状態)

レシピ工房・鍵かけ・レシピパック Pro

  1. レシピ工房: ホットココア/コーヒー/シェイクの3種を、素材ファイルから焼いて .cpsdict に。LLM支援(BYOK)も任意で。
  2. 鍵かけ: フレンドkeyで限定公開パック(.cpmn+menu)を作成。
  3. レシピパック: レシピ同梱の .cprp を作成 — 受け取る側は何も入れずに開けます。
📷スクリーンショット: レシピ工房(焼き上がりログ)

はじめる

入手する、または自分でビルドする

いちばん簡単なのは Store から。開発者はソースを VS2026 で開くだけでビルドできます(外部ライブラリ不要)。

Microsoft Store で入手

Preview は無料で公開中。Standard / Pro は準備中で、当面はソースからビルドできます。

# Store で「CPS Preview」を検索、または
# ストア ID  9MXBCTF6FWBF

ソースからビルド(VS2026)

NuGet も vcpkg も不要。zstd はソース同梱で一緒にコンパイルされます。

# 1. sln を開く（例: Standard）
cps_std_app.sln
# 2. 構成を Release / x64 に
# 3. ビルド → ソリューションのビルド
Ctrl+Shift+B
# => build\cps_std_app.exe
Preview を配布・実行するとき: exe と同じ場所に zdicts\ フォルダ(アイスココアの内蔵辞書6種)を置いてください。exe と zdicts はセットです。

ファイルの種類

拡張子の早見表

  • .cps標準の梱包ファイル。5つの方式のいずれか(開封時は自動判定)。
  • .cpsv瓶づめ。フレーム＋RS で守った頑丈な容器。
  • .cpsdictレイヤー/辞書。レイヤー管理からインストールして使う。
  • .cpgenペアの世代パック。育てた辞書を相手に渡す(系譜ID/世代つき)。
  • .cpp2ペアパック(効率)。ペア辞書で最小に詰めた日常づかい。
  • .cpp3ペアパック(ひみつ)。置換＋4Dシャッフルの限定開示。
  • .cpp3lペアパック(かぎつき)。ひみつ＋1通ごとの鍵の二層。
  • .cprpレシピ同梱の自己完結パック。辞書なしで開ける。
  • .zdictアイスココアの内蔵辞書(6種・同梱)。
